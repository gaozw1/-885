# 框架
**功能：**  冒泡排序

- 【参数类型】    整形数组,整形数组长度
- 【返回值类型】  void
```c
void sort(int array[], int len) {
    for (int i = 0; i < len - 1; i++)            //外部循环控制所有回合
        for (int j = 0; j < len - i - 1; j++) { //内部循环实现每一轮的冒泡处理
            int tmp = 0;
            if (array[j] > array[j + 1]) {
                tmp = array[j];
                array[j] = array[j + 1];
                array[j + 1] = tmp;
            }
        }
}
```
